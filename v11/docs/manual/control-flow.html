6. Control Flow

Conditionals and loops are expressions. Use do { ... } to embed blocks inline; an empty block yields void (truthy). 

message := do {
  total := compute_total()
  `Total: ${total}`
}

6.1 Conditionals

if … or … chains mirror Julia’s if/elseif/else. Branches return their expression value. Only false, nil, and values of type Error are falsy. 

grade = if score >= 90 { "A" }
        or score >= 80 { "B" }
        or { "C or lower" }

6.2 Loops

Loops evaluate to void.

while counter < 3 {
  print(counter)
  counter = counter + 1
}
for { item, idx } in items.enumerate() {
  print(`items[${idx}] = ${item}`)
}
for n in 0...len { total = total + n }

continue skips to the next iteration.

6.3 Loop Expression

loop { ... } is an expression that can yield a value via break: 

first_even = loop {
  if numbers.is_empty() { break nil }
  n = numbers.pop()
  if n % 2 == 0 { break n }
}

6.4 Breakpoints

breakpoint 'label { ... } establishes a named exit point; break 'label value unwinds to it. Plain break exits the innermost loop. 

winner = breakpoint 'scan {
  for cand in candidates {
    if cand.score > threshold {
      break 'scan cand
    }
  }
  nil
}

6.5 Safe Navigation

value?.field and value?.fn() short-circuit to nil if the left side is nil; otherwise they behave like normal access/call.
