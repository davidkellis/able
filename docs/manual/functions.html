4. Functions

Functions are first-class values. The last expression returns implicitly; use return for early exit. 

fn greet(name: string) -> string {
  `Hello ${name}`
}

fn make_adder(delta: i32) -> (i32 -> i32) {
  { value => value + delta }
}

fn find_first_negative(items: Array i32) -> ?i32 {
  for item in items {
    if item < 0 { return item }
  }
  nil
}

4.1 Lambdas & Placeholders

Able offers concise lambda syntax and placeholder lambdas for quick partial application. 

{ x, y => x + y }
fn(x: i32) -> string { x.to_string() }
numbers.map(@ * 2)
add_10 = add(@, 10)

Trailing lambdas mirror Julia’s do blocks:

sum = numbers.reduce(0) { acc, n => acc + n }

4.2 Pipelines

|> forwards values. The right-hand side must reference % or evaluate to a unary callable. 

result = value
  |> %.normalize()
  |> clamp(@, 0, 1)
  |> (@ * 100)
