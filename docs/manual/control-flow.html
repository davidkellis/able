3. Control Flow

Blocks are expressions; the last expression provides the value. do { ... } embeds blocks inline. 

message := do {
  total := compute_total()
  `Total: ${total}`
}

3.1 Conditionals

if … or … chains behave like Julia’s if/elseif/else, returning the chosen branch. Only false, nil, and values of type Error are falsy. 

grade = if score >= 90 { "A" }
        or score >= 80 { "B" }
        or { "C or lower" }

3.2 Loops

while and for loops evaluate to void.

while counter < 3 {
  print(counter)
  counter = counter + 1
}
for {item, idx} in items.enumerate() {
  print(`items[${idx}] = ${item}`)
}
for n in 0...len { total = total + n }

3.3 Breakpoints

breakpoint 'label { ... } establishes an exit point; break 'label value unwinds to it. Plain break targets the innermost loop. 

winner = breakpoint 'scan {
  for cand in candidates {
    if cand.score > threshold {
      break 'scan cand
    }
  }
  nil
}
