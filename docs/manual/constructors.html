7. Constructors & Updates

Struct literals require all fields. Positional structs use tuple-style syntax. Singleton structs carry a single value identical to their type name. 

origin := Point { x: 0, y: 0 }

polar := Polar { radius: 1.0, theta: 0.0 }
polar = Polar { ...polar, theta: pi / 2.0 }

state := Active

Arrays, ranges, channels, and maps rely on helpers in the standard library sketches (see stdlib/ and design/channels-mutexes.md).
