6. Methods

6.1 Inherent Methods

methods Type { ... } attaches behaviour directly to a struct. Inside the block, #field touches the first parameter (conventionally self). 

struct Counter { value: i32 }

methods Counter {
  fn new(start: i32) -> Self { Counter { value: start } }

  fn #increment(by: i32 = 1) -> void {
    #value = #value + by
  }
}

6.2 Interfaces & Implementations

Interfaces capture behaviour; impl blocks provide concrete implementations. Defaults are supported. 

interface Display for T {
  fn to_string(self: Self) -> string
}

impl Display for User {
  fn to_string(self: Self) -> string {
    `User(${self.id}, ${self.name})`
  }
}

Specificity prefers concrete implementations over generic ones. Name an impl to disambiguate. 

Sum = impl Monoid for i32 {
  fn id() -> Self { 0 }
  fn op(self: Self, other: Self) -> Self { self + other }
}

total = Sum.op(2, 3)

Values typed as interfaces dispatch dynamically: 

fn render(value: Display) -> string {
  value.to_string()
}
